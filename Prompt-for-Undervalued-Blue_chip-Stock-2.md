# 优质股低位买入统一分析模板

## 模板定位

本模板用于分析一只股票是否值得在相对低位纳入观察、等待买入或尝试建仓。
请严格按照 stock_comprehensive_analysis_prompt.md 的标准，分析{}是否属于当前可关注的超低位买入标的，并给出完整结论。

目标不是寻找“跌得多”的股票，而是寻找同时满足以下条件的标的：

1. 价格已处于相对低位。
2. 基本面没有出现致命破坏。
3. 大雷可识别、可量化、可跟踪。
4. 下行风险可控，上行空间明确。
5. 买入策略具有可执行性，而不是股评式空话。

本模板适用于：

- 优质股、行业龙头、细分龙头。
- 周期低谷、情绪错杀、估值压缩后的观察与布局。

本模板不适用于：

- ST、退市边缘股。
- 纯题材炒作股。
- 流动性极差、财务真实性存疑的个股。

---

## 一、总原则

### 1. 结论必须唯一

最终结论只能是以下三种之一：

- 可以买
- 观察等待
- 回避

不允许输出：

- 可适当关注
- 逢低布局
- 控制风险即可
- 中长期看好

### 2. 风险优先于收益

先排雷，再谈买点；先算下行，再谈上涨；先定止损，再定仓位。

### 3. 数据优先级

优先使用最新可核验数据：

1. 最新行情
2. 最新公告
3. 最新年报/季报/业绩预告
4. 最新行业数据
5. 最新监管与法律信息

如果拿不到实时数据，必须明确写出：

- 已核验的数据
- 未核验的数据
- 结论属于“可执行判断”还是“参考判断”

### 4. 时间周期统一

默认分析周期为未来 `3-6 个月`。

所有以下内容都按这个周期评估：

- 盈亏比
- 第一目标位与第二目标位
- 下跌 10% / 20% 的概率
- 买入等待期限

---

## 二、信息质量分级

在正式分析前，先给出信息质量等级。

### Level A：可执行

- 核心行情、公告、财报数据齐全。
- 最新财报距离当前不超过 3 个月，或已有可信的一季报/年报更新。
- 重大事件没有明显信息缺口。

说明：可以用于实际交易判断。

### Level B：参考

- 核心数据大体齐全，但部分关键数据缺失。
- 结论需要依赖业绩预告、机构预测或部分推演。

说明：可以用于观察与计划，但不宜直接重仓执行。

### Level C：观察

- 关键财报、公告、诉讼、商誉或流动性数据缺失。
- 重大事项处于待披露或待判定状态。

说明：只能纳入观察池，原则上禁止开仓。

## 三、分析流程

### 第一步：快速排雷（否决项）

若以下任一项为高风险，默认不能给出“可以买”：

1. 审计非标、无法表示意见、保留意见。
2. 证监会立案调查未落地。
3. 控股股东高比例质押且存在平仓风险。
4. 商誉过高且对应资产组经营恶化明显。
5. 连续两年扣非净利润大幅恶化或持续亏损。
6. 短债压力大，货币资金无法覆盖。
7. 重大诉讼、仲裁、专利纠纷、控制权纠纷可能实质影响经营。
8. 面临退市风险或流动性明显枯竭。

快速排雷的目标：

- 不是证明它能涨。
- 而是先判断它会不会出大事。

### 第二步：收集核心信息

至少覆盖以下维度：

#### 1. 价格与趋势

- 最新价
- 近 5 日涨跌幅
- 近 20 日涨跌幅
- 近 52 周高点和低点
- 当前价格在 52 周区间中的分位
- 是否处于下降趋势、横盘筑底或右侧转强
- 成交量、换手率、日均成交额

#### 2. 估值与资产安全垫

- 市值
- 动态 PE / 静态 PE
- PB
- 股息率
- 历史估值中位数对比
- 行业可比公司估值对比

#### 3. 基本面质量

- 最近一期营收、净利润、扣非净利润
- 经营现金流
- 毛利率、净利率、ROE
- 资产负债率
- 有息负债、短债压力
- 货币资金

#### 4. 资产真实性

- 应收账款变化
- 存货变化
- 资产减值损失
- 信用减值损失
- 商誉余额
- 商誉占总资产比例
- 商誉占净资产比例

#### 5. 公司治理与股东行为

- 近 6 个月是否有大股东减持
- 是否有高管增持
- 是否有大额解禁
- 控股股东质押比例
- 再融资、可转债、H 股/境外上市等资本运作压力

#### 6. 法律与合规风险

- 重大诉讼
- 重大仲裁
- 专利纠纷
- 员工持股平台纠纷
- 上市申请相关法律争议
- 海外业务合规与贸易限制
- 是否存在境内或境外监管调查

#### 7. 行业与市场环境

- 所属行业当前景气度
- 行业是否处于估值压缩期
- 当前市场风格是否压制该类股票
- 是否存在政策性利空或政策催化

### 第三步：判断现在是否值得买入

从以下五个维度逐项判断：

1. 价格位置是否够低。
2. 估值是否已具备安全边际。
3. 基本面是否仍有韧性。
4. 风险暴露是否已经出清或可控。
5. 未来 3-6 个月是否存在预期差和催化。

判断原则：

- 低位但基本面持续恶化，不值得买。
- 低位但大雷未证伪，只能观察等待。
- 低位、风险可控、估值偏低、预期差存在，才具备买入资格。

### 第四步：盈亏比与下跌概率测算

必须同时给出以下项目：

1. 预设买入区间
2. 止损位
3. 第一目标位
4. 第二目标位
5. 当前价下跌 10% 的概率
6. 当前价下跌 20% 的概率

并进行以下计算：

- 预期下行空间 = `(买入均价 - 止损位) / 买入均价`
- 第一目标收益 = `(第一目标位 - 买入均价) / 买入均价`
- 第二目标收益 = `(第二目标位 - 买入均价) / 买入均价`
- 当前价下跌 10% 对应价格 = `当前价 × 0.9`
- 当前价下跌 20% 对应价格 = `当前价 × 0.8`
- 保守盈亏比 = `第一目标收益 / 预期下行空间`
- 中性盈亏比 = `第二目标收益 / 预期下行空间`

#### 下跌概率判断必须结合的因素

1. 当前是否处于明确下降趋势。
2. 过去 52 周最大回撤和历史波动特征。
3. 当前价距离关键支撑位有多远。
4. 财报、诉讼、商誉、监管等是否可能触发二次杀估值。
5. 行业板块是否仍在下修周期。

#### 下跌概率统一分档

- 很低：`0% - 15%`
- 较低：`15% - 30%`
- 中等：`30% - 50%`
- 较高：`50% - 70%`
- 很高：`70% - 100%`

#### 参与标准

- 保守盈亏比低于 `1:1.5`，原则上不参与。
- 正常参与标准至少达到 `1:2`。
- 若存在黄灯风险，保守盈亏比至少达到 `1:3`。
- 下跌 10% 概率为“较高”或“很高”，不建议左侧开仓。
- 下跌 20% 概率达到“中等”及以上，必须下调仓位上限。
- 下跌 20% 概率为“较高”或“很高”，原则上不应给出激进买入建议。

### 第五步：大雷排查

逐项给出 `无明显异常 / 有疑点 / 高风险` 结论：

1. 业绩雷
2. 现金流雷
3. 负债雷
4. 商誉雷
5. 审计雷
6. 质押减持雷
7. 监管诉讼雷
8. 流动性雷
9. 境外法律纠纷雷
10. 行业雷

排雷规则：

- 只要存在一个足以破坏逻辑的高风险项，就不能给“可以买”。
- 商誉占总资产超过 `15%` 或占净资产超过 `25%` 时，必须单独分析来源、资产组、减值触发因素。
- 存在重大境外诉讼、仲裁或股权纠纷，且可能影响经营、融资、上市、控制权稳定性时，风险至少上调一档。

### 第六步：制定买入策略

必须明确写出以下内容：

1. 推荐策略类型：左侧试仓 / 右侧确认 / 分批低吸 / 放弃等待。
2. 第一笔买入条件。
3. 第二笔加仓条件。
4. 第三笔加仓条件。
5. 总仓位上限。
6. 止损执行条件。
7. 放弃观察条件。

#### 默认策略框架

- 全绿灯、下跌概率低：可以分批低吸或右侧确认。
- 有黄灯、趋势未稳：小仓试错，等待确认后再加。
- 下跌 20% 概率中等及以上：总仓位必须下调。
- 存在重大不确定性：以观察等待为主。

#### 建议仓位分三段

1. 第一笔：计划仓位的 `20%-30%`，只用于试错。
2. 第二笔：确认止跌或基本面改善后，再补 `30%-40%`。
3. 第三笔：趋势和逻辑都确认后，再补足目标仓位。

### 第七步：等待买入期间重点关注

至少跟踪以下项目：

1. 最新财报或业绩预告是否改善。
2. 是否有突发公告：减持、诉讼、质押、立案、再融资。
3. 行业景气数据是否改善。
4. 股价是否形成止跌结构。
5. 成交量是否由恐慌放量转为缩量企稳。
6. 北向、机构、龙虎榜资金是否出现承接信号。
7. 商誉对应资产组是否继续恶化。
8. 境内外法律纠纷是否有新进展。

等待期的核心不是盯价格，而是回答四个问题：

- 风险有没有继续释放？
- 利空有没有钝化？
- 资金是否开始承接？
- 基本面是否接近拐点？

## 四、判断口径

### 可以买

只有大体同时满足以下条件，才能输出“可以买”：

1. 股价处于阶段低位或历史相对低位。
2. 估值不贵，至少不显著高于行业可比。
3. 基本面虽弱但未崩，或已经出现改善迹象。
4. 没有足以致命的财务、审计、监管、退市类风险。
5. 盈亏比至少达到 `1:2`。
6. 有明确催化或修复逻辑。
7. 下跌 10% 和下跌 20% 的概率不能明显偏高。

### 观察等待

出现以下情况之一，优先输出“观察等待”：

1. 价格已低，但趋势还没止稳。
2. 盈亏比尚可，但大雷还未证伪。
3. 基本面可能在磨底，但拐点证据不够。
4. 行业压制仍强，短期缺乏催化。
5. 当前价继续下跌 10% 或 20% 的概率仍然不低。

### 回避

出现以下情况之一，直接输出“回避”：

1. 财务质量明显恶化。
2. 审计、监管、诉讼、退市风险高。
3. 主营长期失血，没有反转证据。
4. 估值不低且仍在下修通道。
5. 盈亏比明显不划算。
6. 商誉规模过大且减值风险无法合理评估。
7. 存在可能实质影响融资、境外上市、海外销售或控制权稳定性的重大法律纠纷。
8. 从当前价下跌 20% 的概率较高，而风险回报补偿明显不足。

## 六、输出风格要求

1. 先给结论，再给证据。
2. 必须明确，不允许模糊表达。
3. 如果数据不足，直接说不足，不要假装精确。
4. 所有建议必须可执行，必须包含价格、概率、仓位或触发条件。
5. 尽量优先选择行业龙头与流动性好的标的，不轻易为“便宜”牺牲质量。
